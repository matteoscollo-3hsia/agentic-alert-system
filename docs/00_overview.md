# Overview

Questo repository ospita un MVP di Agentic Alert System per lead generation B2B.

## Concetto
- Input: database di aziende, trigger di vendita, e fornitori di notizie.
- Processo: ricerca giornaliera di notizie e matching con trigger.
- Output: alert con nome azienda, trigger rilevato e contact owner.
